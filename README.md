🏥 Hospital Management System

The Hospital Management System (HMS) is a web-based application developed to automate and manage the daily operations of a hospital. It provides a centralized platform to handle patient records, doctor management, appointment scheduling, and billing, reducing manual work and improving efficiency.

The system supports role-based access for Admin, Doctor, and Patient, ensuring secure and structured management of hospital data.

🚀 Features

Patient registration and medical record management

Doctor and staff management

Appointment scheduling system

Billing and payment management

Role-based authentication (Admin / Doctor / Patient)

Secure database storage

User-friendly web interface
